AWS CLI
-------
1.Login to EC2 instance
*ssh ubuntu@IPADDRESS
-> In cmd 
first make authorized the pem file(key value pair)
*chmod 600 /filepath/test.pem
*ssh -i /filepath/test.pem ubuntu@IPADDRESS

->After installing AWS CLI use *aws version you get some message thats means cli is installed.
->For proper connection use security keys to connect
*aws configure then add all the security things and authenticate the EC2 instace
->whatever u enter in the cmd it will reflect in EC2 instance.

->To create S3 bucket in CLI 
*aws S3 mb babul123

---------------
1.ls -ltr -> details with list of file in pwd
2.#!/bin/bash -> shebang

4.set -x #debug mode(show the command whats it executing also print the output)
5.ps -ef #print the processes
6.ps -ef | grep "amazon" #use string to show process detail 
 #| pipe command send the output from first to second command
7.date | echo"date is"

8.awk #give the specific column
 ps -ef | grep amazon | awk -F" "'{print $2}'

10.set -o #pipefail
